<h1>🎬 MAX - Canal por Assinatura</h1>

<h2>Projeto desenvolvido como uma landing page de streaming, inspirado em plataformas como HBO Max.

A aplicação simula um site de assinatura de serviço de streaming, apresentando uma interface moderna com foco em design visual e experiência do usuário.</h2>

<br>

<h1>🚀 Sobre o Projeto</h1>

<h2>A página apresenta:
  <br>
  <br>
- 🎥 Tela inicial com chamada para assinatura
  <br>
- 🎨 Layout inspirado em plataformas de streaming
  <br>
- 📱 Interface responsiva
  <br>
- 🖥️ Organização visual moderna
  <br>
- 🔄 Navegação simples e intuitiva</h2>

<br>

<h1>🛠️ Tecnologias utilizadas</h1>

<h2>- HTML5
  <br>
- CSS3
  <br>
- JavaScript</h2>
